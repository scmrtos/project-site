
**scmRTOS** &nbsp;– это компактная операционная система (ОС) реального времени, предназначенная для использования в микроконтроллерах (МК). 

**scmRTOS** способна выполняться на совсем крошечных МК, имеющих объём оперативной памяти порядка 512 байт. Она написана на языке программирования С++ и поддерживает [ряд платформ](platforms.md). Подробное описание ОС, включая [интерактивную документацию](preface.md), приведено в следующих разделах сайта.

Исходный код ОС может быть склонирован или загружен с сайта [github.com](https://github.com/scmrtos/scmrtos). Там же можно найти [ряд примеров](https://github.com/scmrtos/scmrtos-sample-projects), которые иллюстрируют использование ОС и могут служить отправной точкой для рабочих проектов.

Полный архив, включающий исходный код ОС, примеры, документацию[^1] и т.п. также может быть загружен с [соответствующей страницы проекта на github.com](https://github.com/scmrtos/scmrtos/releases/).

[^1]: Данный архив включает в себя PDF версии документации. Этот вариант является устаревшим и не рекомендуется для использования. Вместо него предлагается [онлайн вариант](preface.md), как более удобный, оперативный и содержащий описание всех актуальных изменений.
